Detailed Description

subclass representing a list of nodes in the syntax tree, layed out horizontally

This type of node also implements typesetting sub-/superscript above/below the previous node (if JKQTMathTextNode::is() is true for that previus node. The drawing is done as defined in the following image:

In addition this node also ensures that subscripts are moved to the left for italic previous text or nodes that require such a shift (e.g. \nabla or \int ):

This is implemented using JKQTMathTextNodeSize::baselineXCorrection .

In addition this class implements execution of instructions derived from JKQTMathTextModifyEnvironmentNodeMixIn.
