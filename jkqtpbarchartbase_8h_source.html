33/** \brief This is a base-class for all bar graphs with vertical or horizontal orientation (the orientation is implemented in dervied classes!)
34 * \ingroup jkqtplotter_barcharts
35 *
36 * This class plots a bargraph. This image explains the parameters:
37 *
38 * \image html bargraph_basics.png
39 *
40 * By default the sift parameter is, so the bar is centered at the x-value. The width is 0.9,
41 * so adjacent bars are plotted with a small distance between them. It is possible to use these two parameters
42 * to plot multiple bars for every x-value, by having on JKQTPSpecialLineHorizontalGraph object per
43 * set of bars that belong together. For example for three bars per x-value one would set:
44 * \verbatim
45 * width=0.3
46 * shift=-0.3 / 0 / +0.3
47 * \endverbatim
48 * This results in a bargraph, as shown here:
49 *
50 * \image html JKQTPBarVerticalGraph.png
51 *
52 * You can also set JKQTPBarGraphBase::FillMode::TwoColorFilling, which uses different fill styles for bars above and below
53 * the baseline of the graph:
54 *
55 * \image html JKQTPBarVerticalGraphTwoColorFilling.png
56 *
57 * If you use JKQTPBarGraphBase::FillMode::FunctorFilling you can specify the fill style by a functor, e.g.
58 * \code
59 * graph->setFillMode(JKQTPBarGraphBase::FillMode::FunctorFilling);
60 * graph->setFillBrushFunctor(
61 * [](double key, double value) {
62 * return QBrush(QColor::fromHsvF(key/12.0, 1.0, 1.0));
63 * }
64 * );
65 * \endcode
66 *
67 * The result may look like this:
68 *
69 * \image html JKQTPBarVerticalGraphFunctorFilling.png
70 *
71 * You can also completely customize the drawing by defining a custom draw functor:
72 * \code
73 * graph->setCustomDrawingFunctor(
74 * [](JKQTPEnhancedPainter& painter, const QRectF& bar_px, const QPointF& datapoint, Qt::Orientation orientation, JKQTPBarGraphBase* graph) {
75 * // draw the bar (if required), pen and brush are already set properly
76 * painter.drawRect(bar_px);
77 * // now we can add some decoration or replace the instruction above:
78 * // ........
79 * }
80 * );
81 * // enable usage of cutom draw functor
82 * graph->setUseCustomDrawFunctor(true);
83 * \endcode
84 *
85 * See \ref JKQTPlotterBarchartsCustomDrawFunctor for a detailed example.
86 *
87 * The result may look like this:
88 *
89 * \image html JKQTPBarVerticalGraphCustomDrawFunctor.png
90
91 *
92 * \see JKQTPBarHorizontalGraph, JKQTPBarVerticalGraph
93 */
94class JKQTPLOTTER_LIB_EXPORT JKQTPBarGraphBase: public JKQTPXYBaselineGraph, public JKQTPGraphLineStyleMixin, public JKQTPGraphFillStyleMixin {
