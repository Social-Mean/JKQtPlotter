56/*! \brief represents a font specifier for JKQTMathText. The font consists of two parts: the actual font and the font used for math output (which may be empty)
57 \ingroup jkqtmathtext_tools
58
59 \section JKQTMathTextFontSpecifier_specialSyntax Font Definition Syntax
60 JKQTMathTextFontSpecifier::fromFontSpec() defines a special syntax that when parsed allows to seth the font,
61 a math-mode alternative:
62 - \c FONTNAME : set the text-mode font only
63 - \c FONTNAME+MATH_FONTNAME : set the text- and math-mode fonts
64
65
66
67
68 \section JKQTMathTextFontSpecifier_specialNames Special Font Names
69 This object also implements replacing special font names with actual fonts. Supported special font names are:
70 - \c default / \c app / \c application - the applications default font
71 - \c application-sf - a font for "sans", based on the application font
72 - \c times / \c serif - a general serif font
73 - \c sans-serif - a general sans-serif font
74 - \c typewriter - a general typewrter/monospaced font
75 - \c cursive
76 - \c decorative
77 - \c fantasy
78 - \c monospace
79 - \c system
80 .
81
82 If copiled with Qt>5.3 you can also use these:
83 - \c fixed
84 - \c smallest_readable
85 - \c title
86 - \c general
87 .
88
89 Also some sepcial fonts are defined:
90 - xits XITS fonts
91 - stix STIX fonts
92 - asana ASANA fonts
93 - fira Fira fonts
94*/
